#### problem: a long chain on the same level abstraction

```ts
CatsList
-> (render -> (
    (renderContent -> (
      (renderCatsList -> (
        (listItem -> JSX)
        + JSX))
      + JSX))
    + JSX))
```

---

#### step 1: inline

```ts
CatsList -> render -> JSX
```

---

#### step 2: introduce indirection bottom-up

```ts
CatsList
-> (render -> (
     (CatsListItem -> JSX)
     + (OutOfCatsButton -> JSX)
     + JSX
   ))
```
